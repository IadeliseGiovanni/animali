🚀 PetFlow: Adozioni Animali API🐾 ProgettoPetFlow è una piattaforma Full Stack progettata per digitalizzare e ottimizzare il processo di adozione degli animali. Il sistema permette di gestire il ciclo di vita dell'adozione: dall'inserimento dell'animale nel database fino alla gestione degli eventi e degli adottanti, garantendo sicurezza e integrità dei dati tramite autenticazione JWT.🛠️ Architettura e MeccanismoIl progetto è diviso in due ecosistemi indipendenti che comunicano tramite API REST:Core Backend: Sviluppato in IntelliJ IDEA, gestisce la logica di business, la persistenza su database relazionale e la sicurezza dei dati.User Interface: Sviluppata in VS Code, offre un'esperienza utente reattiva e moderna, consumando i servizi esposti dal backend in modo asincrono.⚙️ Configurazione Rapida🗄️ Database SetupIl sistema è configurato per essere "Plug & Play".Crea un DB vuoto chiamato adozioni in PostgreSQL.Grazie alla configurazione ddl-auto=update, le tabelle vengono modellate automaticamente all'avvio.Configurazione URL:spring.datasource.url=jdbc:postgresql://localhost:5432/adozioni🚀 Avvio ApplicazioneLayerIDEComandoPortaBackendIntelliJRun AnimaliApplication8080FrontendVS Codeng serve4200
